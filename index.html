Política de Privacidad

Esta política describe cómo recopilamos, utilizamos y protegemos tu información en Impostor Mobile.

Última actualización: 6 de marzo de 2026

Al utilizar este juego multijugador, aceptas las prácticas detalladas en este documento. Hemos diseñado Impostor Mobile limitando la recolección de datos al mínimo indispensable para que el juego funcione.

1. Desarrollador y Responsabilidad

Impostor Mobile es un proyecto independiente desarrollado de manera individual. No existen vinculaciones corporativas ni con agencias externas para la gestión de datos de los usuarios.

2. Recopilación y Uso de Datos

Para garantizar la funcionalidad del juego multijugador en tiempo real y la seguridad de los usuarios, tratamos los siguientes datos:

Alias (Apodo)

Se utiliza únicamente para identificar al jugador temporalmente dentro de las salas de juego. No se requiere tu nombre real ni vinculación con cuentas de correo o redes sociales.

Mensajes de Chat

Los mensajes enviados en las salas se almacenan temporalmente en nuestros servidores para permitir la interacción del juego, y son descartados una vez que dejan de ser útiles.

IDs de Dispositivo

Se recopilan identificadores técnicos (como el ID de publicidad de Google) necesarios para el funcionamiento de los servicios de anuncios y analíticas integrados.

3. Divulgación de Datos a Terceros

Declaramos de forma transparente el uso de los siguientes proveedores de servicios que tienen acceso a datos técnicos:

Google AdMob

Utilizamos anuncios. AdMob utiliza SDKs de terceros para mostrar banners e intersticiales, pudiendo recopilar datos del dispositivo para la personalización y medición publicitaria.

Supabase

Servicio de infraestructura backend utilizado para gestionar las salas, la lógica en tiempo real y el almacenamiento temporal del chat.

Expo

Framework base de la aplicación que puede recopilar datos técnicos anónimos para mejorar la estabilidad.

4. Permisos y Seguridad

Permiso de Acceso a Internet

Es el único permiso requerido. Necesario para sincronizar partidas, el chat multijugador y cargar los anuncios. No solicitamos permisos invasivos o de alto riesgo.

Tránsito Seguro

Toda la información se transmite de forma encriptada (HTTPS/WSS) para proteger tu privacidad frente a terceros.

5. Almacenamiento Local

La aplicación utiliza almacenamiento local seguro en tu dispositivo (mediante Expo Secure Store / Async Storage) para:

  • Mantener tu sesión activa.
  • Guardar tus preferencias, como el idioma seleccionado.
  • Almacenar el historial reciente para mejorar la experiencia de usuario.

Estos datos permanecen estrictamente en tu dispositivo y no se envían a servidores externos.

6. Público Objetivo y Menores

Dado que la aplicación incluye una función de chat de texto libre y publicidad servida por terceros, el público objetivo principal son personas mayores de 13 años.

No recopilamos intencionadamente datos personales de niños menores de 13 años. Se exige la supervisión constante de un adulto y el cumplimiento de las políticas locales (COPPA/GDPR-K).

Si tomamos conocimiento de que hemos recopilado datos de un menor sin consentimiento, los eliminaremos inmediatamente.

7. Transparencia, Derechos y Contacto

Me comprometo a mantener esta política actualizada. Tienes derecho a solicitar la eliminación de cualquier dato que podamos tener asociado a ti.
